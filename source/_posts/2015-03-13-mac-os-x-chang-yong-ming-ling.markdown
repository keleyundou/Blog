记录下平时在Mac终端常用的指令

当前工作目录

```
pwd
```

进入文件夹

```
cd 文件名
```

上级目录

```
cd ..
```

返回root

```
cd ~
```
<!--more-->

返回上一个访问的目录

```
cd -
```

删除

```
rm -r 文件名
```

列出目录下所有文件

```
ls
```

将文件拷贝到目标目录下

```
cp 文件名 目标目录
```

新建文件夹

```
mkdir 文件夹名
```
